# 🎨 Kelly - Professional Portfolio & Resume Template

A modern, fully responsive, and clean HTML/CSS template designed for creative professionals, illustrators, graphic designers, and web developers to showcase their work, skills, and experience beautifully.

---

## ✨ Features

- 📱 **Fully Responsive:** Looks perfect on Mobile, Tablet, and Desktop devices.
- 🧭 **Interactive Navigation:** Smooth scrolling and a fully functional JavaScript dropdown menu.
- 🎯 **Portfolio Grid:** Masonry-style layout with beautiful hover overlay effects.
- 📊 **Animated Skill Bars:** CSS-only animated progress bars that trigger on load.
- 📄 **Complete Pages:** Includes Home, About, Resume, Services, Portfolio, and Contact pages.
- 🎨 **Easy to Customize:** Built with CSS Variables (`:root`) for quick color and font changes.
- ⚡ **Performance Optimized:** Preconnected Google Fonts and optimized asset loading.

---

## 🛠️ Technologies Used

- **HTML5** (Semantic markup)
- **CSS3** (Flexbox, Grid, Animations, CSS Variables)
- **Vanilla JavaScript** (Dropdown toggle, dynamic year update)
- **Font Awesome 6** (For scalable vector icons)
- **Google Fonts** (Roboto, Raleway, Poppins)
